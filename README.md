<h1>Basic HTML Website Project</h1>
<h2>Key Requirement for this:</h2>
<p>In this project, you are required to create a simple HTML-only website with multiple pages. The website should have the following pages:</p>
<ul>
  <li>Homepage</li>
  <li>Projects</li>
  <li>Articles</li>
  <li>Contact</li>
</ul>
<p>The website should have a navigation bar that should be present on all pages and link to all the pages.</p>
